# Scope

## Разрешено
- создавать новые файлы;
- редактировать файлы только внутри `rewrite_v2/`;
- проектировать V2-архитектуру звонков и уведомлений;
- писать чистые интерфейсы, state-machine, DTO, сервисы, схемы событий.

## Запрещено
- менять что-либо вне `rewrite_v2/`;
- править legacy-файлы в `frontend/lib/services/`, `frontend/lib/app/`, `src/call/`, `src/notifications/`;
- удалять старый код;
- менять production flow;
- трогать `main`-ветку;
- менять БД, prisma schema, env, pm2, серверные скрипты.

## Формат работы
- работать маленькими шагами;
- не делать интеграцию;
- писать код так, чтобы потом его можно было перенести в основной проект почти без переписывания.
